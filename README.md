# RoboSpeaker

RoboSpeaker is a tool that turns text into speech. It was made by Mohammed Fariq using Python. This tool is, for Windows. It uses the things that come with Windows like PowerShell and the Speech API. You do not need to be connected to the internet for it to work. You just type what you want to hear in the console and RoboSpeaker says it loud right away.

---

##

* **It is all Windows:** RoboSpeaker uses the os library that comes with Python so you do not need to add anything

* **It works without the internet:** You can use RoboSpeaker even when you are not connected to the internet.

* **It keeps asking for input:** RoboSpeaker keeps asking you for text to speak until you tell it to stop.

---

##

* **You need Windows:** You need to have Windows 10 or Windows 11 to use RoboSpeaker.

* **You need Python:** You need to have Python 3.x to use RoboSpeaker.

* **You can use PyCharm:** PyCharm is a tool to use with RoboSpeaker.

---

##

1. Open PyCharm.

2. Make a file called main.py and put the RoboSpeaker code in it.

3. Run the script.

4. Type what you want RoboSpeaker to say and press Enter.

5. Type q and press Enter to make RoboSpeaker stop.
